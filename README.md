# Healthcare Readmission Prediction AI Model

This project predicts whether a patient will be readmitted to a hospital using machine learning.

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML, CSS, JavaScript

## Project Structure
- backend: Flask API + ML model
- frontend: Web interface

## How to Run
1. Install requirements
2. Run backend (app.py)
3. Open frontend predict.html
